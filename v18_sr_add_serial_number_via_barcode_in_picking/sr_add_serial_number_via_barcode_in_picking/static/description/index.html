Add Serial Number Via Barcode On Incoming and Outgoing Pickings/Shipments

It will take more time to scan serial/lot number of the product while incoming shipement or outgoing shipment because you have to perform manual process for that. This module will automate this process and more easier to save your most of the time. with this app you can scan your serial/lot number via barcode scanner as possible as fast you can, serial number will add in system if its incoming shipment and linked with existing serial/lot number if its outgoing shipment.

Module Features
  • automate process for scan serial/lot number for picking
  • Easy to scan serial/lot number for products in incoming and outgoing shipments
  • Auto create serial number for incoming shipment
  • Link with existing serial number for outgoing shipment
  • 50 days free support

Incoming Shipment Flow

First of all, you need to configure product as per below screenshot.

Now, create 1 purchase order and confirm. it will create 1 incoming shipment.

In that incoming shipment, you just need to click on three lines to enter serial numbers for that products. shown in below screenshot.

It will open below screen to enter serial/lot number for that products

here, you can see how this process works. you just need to scan barcode one by one and it will add serial/lot number in lines with done quantity.

Once you validate that picking at that time it will create that serial number in system for further use.

Outgoing Shipment Flow.

Create 1 sales quotation and confirm. it will create 1 outgoing shipment.

In that Outgoing shipment, you just need to click on three lines to enter serial numbers for that products. shown in below screenshot.

It will open below screen to enter serial/lot number for that products

here, you can see how this process works. you just need to scan barcode one by one and it will add serial/lot number in lines with done quantity if existed in below lines.
