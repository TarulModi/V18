Generate single customer invoice flow from multiple outgoing shipments

Create 2 or more sales order and confirm it.

Here, You can see delivery orders. Select orders and go to action where you will see menu to generate single invoice against that delivery orders.

When you click on that menu, it will open 1 pop where you can confirm the customer name and journal as well you can enter invoice date and payment term for that invoice.

Here, you can see the generated single invoice against that delivery orders

Generate single vendor bill flow from multiple incoming shipments

Create 2 or more purchase order and confirm it.

Here, You can see incoming shipments. Select shipments and go to action where you will see menu to generate single invoice against that incoming shipments.

When you click on that menu, it will open one popup where you can confirm the vendor name and journal as well you can enter bill date and payment term for that bill.

Here, you can see the generated single vendor bill against that incoming shipments

Support Different Warnings

If you select any invoiced shipment then you will get warning.

If you select different partner's shipments at that time you will get warning.

If you select incoming and outgoing both shipment at a time then you will get warning.

If you select any picking which is not in ready or done state then you will get warning.
