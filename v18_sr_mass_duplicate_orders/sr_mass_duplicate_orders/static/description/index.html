Mass/Bulk Duplicate Records

Currently, Odoo offer duplicate single record at a time. This application will help you to duplicate multiple/bulk record on single click. Simply you just select multiple record from tree view and click on mass duplicate orders from action menu. System will create duplicate record of the selected records. This application works for Sales order, Invoices and Purchase orders.

Mass/Bulk Duplicate Sales Order Flow

Select single or multiple records of Sales Order from Tree view and Go to the action menu and click on Mass/Bulk Duplicate Sales Orders menu

It will Open Wizard.

You can see duplicated Sales order.

Mass/Bulk Duplicate Purchase Order Flow

Select single or multiple records of Purchase Order from Tree view and Go to the action menu and click on Mass/Bulk Duplicate Purchase Orders Menu

It will Open Wizard.

You can see duplicated Purchase order.
