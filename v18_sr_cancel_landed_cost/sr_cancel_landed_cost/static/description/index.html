Cancel and Reset Stock Landed Cost in Odoo

This module will allow you to cancel/reverse stock landed cost even if it's validated. By default odoo don't allow users to cancel/reverse/reset stock landed cost to change the calculations and other stock landed cost details. The main purpose of this application is to rectify the mistake of the user. When you cancel stock landed cost, it will reverse all the calculations i.e cancel related journal entry, reverse the calculation from stock valuations.
